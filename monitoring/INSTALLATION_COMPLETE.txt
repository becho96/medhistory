📊 Что установлено:
═══════════════════════════════════════════════════════════════════

✅ Prometheus - Сбор и хранение метрик
✅ Grafana - Визуализация (3 готовых дашборда)
✅ Custom Exporter - Бизнес-метрики из БД
✅ PostgreSQL Exporter - Метрики базы данных
✅ MongoDB Exporter - Метрики MongoDB
✅ Node Exporter - Системные метрики

📈 Отслеживаемые метрики:
═══════════════════════════════════════════════════════════════════

БИЗНЕС-МЕТРИКИ:
  • Пользователи (всего, активные, новые)
  • Документы (всего, по типам, новые)
  • AI Интерпретации (всего, успешные, с ошибками)
  • Отчеты (всего, новые)
  • Хранилище (объем, количество объектов)

ТЕХНИЧЕСКИЕ МЕТРИКИ:
  • API (RPS, латентность, ошибки)
  • Система (CPU, RAM, Disk, Network)
  • PostgreSQL (соединения, транзакции, размер БД)
  • MongoDB (операции, коллекции, производительность)

🚀 Быстрый старт:
═══════════════════════════════════════════════════════════════════

1. Запустите систему:
   ./scripts/local/start.sh

2. Откройте Grafana:
   http://localhost:3000
   Логин: admin
   Пароль: admin

3. Выберите дашборд:
   • Бизнес метрики
   • Системный обзор
   • Производительность API
